Responsive Open Learning Environments (ROLE) SDK - Sample Gadgets

Version: ${project.version}

This directory contains a set of widgets supporting your work in developing new widgets based on ROLE technologies. In order to use these widgets, simply add the ones you need to a space you use for development.

Contents

  • openapp - sample widgets demonstrating the use of OpenApp services & libraries (user, space, resource).
  • iwc - sample widgets demonstrating the use of ROLE Interwidget Communication incl. a tutorial.
  • cam - sample widgets demonstrating the use of the bundled CAM service (monitoring and querying contextualized attention metadata).
  • pleshare - sample widgets demonstrating the use of the PLEShare Web service (activity pattern sharing).
