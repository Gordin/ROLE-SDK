Responsive Open Learning Environments (ROLE) SDK - Development Widgets

Version: ${project.version}

This directory contains a set of widgets supporting your work in developing new widgets based on ROLE technologies. In order to use these widgets, simply add the ones you need to a space you use for development.

Contents

  • ROLE SDK Service Test Widget - a widget providing a quick overview of the availability of the different ROLE services. Only checks availability when loaded.
  • ROLE IWC Teststub Widget - a widget displaying occurred ROLE IWC intents as reversely ordered list and includin a simple interface for emulating intents.
  • OpenApp Event Viewer Widget - a simple widget listing occurred OpenApp events.
