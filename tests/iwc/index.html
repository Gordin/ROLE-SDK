ROLE Interwidget Communication (IWC) Demonstrator

Bookmark this on Delicious

On this page we demonstrate multi-user-multi-browser remote interwidget communication in a set of widgets created for remote realtime collaboration scenarios among learners.

Prerequisites

Your browser must support the WebSocket API. We recommend Google Chrome.

Instructions

  1. Open multiple different browser instances with this page.
  2. In each browser instance login with a separate account in the "IWC Proxy" Widget. Create accounts in the "IWC Proxy" widget.
  3. Interact with the widgets in any browser instance and see what is happening in the other browser instances.

Available Widgets

  • IWC Proxy - gateway for outgoing/incoming interwidget communication
  • IWC Video Player - watch videos together in real-time: Seek to a position and push "Publish Seek" to broadcast.
  • IWC Map - navigate maps together in real-time: Zoom & pan the map to broadcast navigation.
  • IWC Paint - paint together in real-time: Set color and linewidth. Paint on the canvas to broadcast paint operations.
